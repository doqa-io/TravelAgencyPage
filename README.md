# Travel Agency App

Welcome to the Travel Agency App! This simple web page allows users to explore travel destinations, view available travel packages, and discover top itineraries with beautiful images.

## Features
- List of travel packages (Group Travels, Private Tours)
- Highlighted top itineraries with images and captions
- Clean and accessible HTML structure

## Usage
Open `index.html` in your web browser to view the Travel Agency App page.

## Project Structure
- `index.html` – Main HTML file containing the travel agency content
- `README.md` – Project overview and instructions

## Customization
You can update the travel packages, itineraries, and images by editing the `index.html` file.
